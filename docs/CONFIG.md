# Configuration Reference

> Project: Kaison Sentinel — offline-first build. See START_HERE.md for current setup.
> Last updated: 2026-01-30

- `config/options_master.json` — core options and tool allowlists
- `config/options_override.json` — runtime overrides (proxy, etc.)
- `config/policy.json` — RBAC and defaults
- `config/api_auth.json` — API key auth
- `config/api_limits.json` — rate limiting
- `config/retention.json` — retention policy
- `config/webhooks.json` — webhook endpoints
- `config/report_formats/` — report templates
- `config/mitigation_templates.json` — mitigation patterns
- `config/mitre_attack.json` — MITRE planner techniques (defensive, read-only)
- `config/mitre_report_formats/` — MITRE plan report templates
- `config/playbooks/` — playbook definitions used by MITRE planner and plan wizard
